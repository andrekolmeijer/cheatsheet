André's Cheat Sheet

CSS Selectors

Selector Example Description
Type p { color: green; } Matches the type of the element.
Universal * { font-family: Verdana; } The universal selector selects all elements of any type.
Class .title { color: teal; } In the example, the class is title, so the CSS selector for it is .title.
ID #article-title { font-family: cursive; } The id name is large-title, therefore the CSS selector for it is #large-title.
Attribute [href]{ color: magenta; } In the example: [href] would target all elements with an href attribute and set the color to magenta.
img[src*='winter'] { height: 50px; } The ruleset looks for an img element with an attribute of src that contains the string 'winter', and sets the height to 50px.
Pseudo-class p:hover { background-color: lime; } In the example, whenever the mouse hovers over a paragraph element, that paragraph will have a lime-colored background.

CSS Specificity

Concept HTML CSS Description
Specificity <h1 class='headline'>Breaking News</h1> h1 { color: red; } .headline { color: firebrick; } In the example code, the color of the heading would be set to firebrick, as the class selector is more specific than the type selector.
Chaining <h1 class='special'>Special Heading</h1> h1.special { font-family: Tahoma; } The code would select only the <h1> elements with a class of special. If a <p> element also had a class of special, the rule in the example would not style the paragraph.
Descendant Combinator <ul class='main-list'> <li>1</li> <li>2</li> <li>3</li> </ul> .main-list li { font-weight: bold; } In the example above, .main-list selects the element with the .main-list class (the <ul> element). The descendant <li>‘s are selected by adding li to the selector, separated by a space.
Chaining and Specificity p { color: blue; } .main p { color: red; } Adding more than one tag, class, or ID to a CSS selector increases the specificity of the CSS selector.
Multiple Selectors h1, .menu { font-family: Georgia; } By separating the CSS selectors with a comma, both the <h1> elements and the elements with the menu class will receive the font-family: Georgia styling.

Terminal Commands

Command Description
mkdir Creates a new directory in the working directory.
touch Creates a new file inside the working directory.
echo Outputs the strings that are passed to it as arguments.
